05 — Background Gradients

Background gradients

Four official background tokens. The two CSS gradients work as generic light/dark fallbacks; the two image gradients are pinned to the Main and Odyssey themes. Apply only to body or the top-level container — never to individual components.

Main
bg-main-solid
Clean flat surface — no gradient. Resolves to the theme's --bg token so the page color follows the theme.
Dark
bg-dark
Clean flat surface — no gradient. Resolves to the theme's --bg token so the page color follows the theme.
Agentic AI
bg-main-mesh
Custom mesh image — soft whites, pale blues, and a periwinkle pocket. File: assets/main-theme-bg.png.
Odyssey
bg-odyssey-aurora
Custom aurora image — deep indigo to luminous violet with a warm magenta lift. File: assets/odyssey-theme-bg.png.

07 — Components

Cards (3-column grid)

Standard card pattern at three columns. Hover for elevation. Use this layout for feature lists, integration grids, or product capabilities.

Lightning fast

Sub-100ms response across every endpoint, backed by a global edge network and aggressive caching.

◆
Token-driven

Composable

Mix and match primitives to build the surfaces you need without writing a single off-palette color or off-scale size.
